# MediatekDocuments

Cette application permet de gérer les documents (livres, DVD, revues) d'une médiathèque. Elle a été codée en C# sous Visual Studio 2019. C'est une application de bureau, prévue d'être installée sur plusieurs postes accédant à la même base de données.
L'application exploite une API REST pour accéder à la BDD MySQL.

[Des explications sont données sur la page du projet, ainsi que le lien de récupération.](https://www.nicolasfrere.fr/pages/mediatekDocuments.html)

[Repo de l'API](https://github.com/Gotlub/rest_mediatekdocuments)

![App Screenshot](https://www.nicolasfrere.fr/ressources/MediatekDocumentsCapture.jpg)


Le projet est un fork de [MediaTekDocuments](https://github.com/CNED-SLAM/MediaTekDocuments), auquel des fonctionnalités telles que la modification des livres, des DVD et des revues, la gestion des commandes, des abonnements ont été ajouté .
